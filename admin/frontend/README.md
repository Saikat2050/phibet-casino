# Sweepstake Admin Frontend

Node.js based project (Sweepstake Admin Frontend) provides a solid foundation for building scalable and efficient admin interfaces. Designed with performance and modularity in mind, it offers essential tools and architecture, enabling developers to swiftly create and deploy feature-rich, intuitive admin solutions.

## Installation

- Node JS version Used is >=20.15.0.

## How to run application in Dev mode (locally)

- First copy .env.sample file to .env and fill the all env variables (NODE_ENV = 'development')
- Then run `npm i` to install all the dependencies
- After installing, please run `npm run dev`. It will start the project in watch mode

## How to run application in production mode

- First copy .env.sample file to .env and fill the all env variables (NODE_ENV = 'production')
- Then run `npm i` to install all the dependencies
- After installing, please run `npm run preview` to build and start the project

## Other NPM scripts and Make Commands

- `npm run lint` for linting of the application
- `make clean` for deleting the project build
- `make push` for push the changes to the git stream
- `make pull` for pull the latest changes from the git stream with rebase
